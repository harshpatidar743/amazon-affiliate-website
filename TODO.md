# Amazon Affiliate Website - Project Plan

## Project Overview ✅
- Create a single-page Amazon affiliate website
- Easy product management via JSON file
- General products (multi-niche support)
- Clean, modern design

## Files Created ✅
1. ✅ **products.json** - Product data file (easy to edit)
2. ✅ **index.html** - Main single page website
3. ✅ **styles.css** - Styling and design
4. ✅ **script.js** - Product loading and interactions
5. ✅ **README.md** - Instructions for user

## Features Implemented ✅
- Responsive product grid
- Product cards with image, title, price, description
- "Buy on Amazon" buttons with affiliate links
- Easy JSON-based product management
- Clean header and footer sections
- Category filtering
- Mobile responsive design

## How to Use
1. Open index.html in your browser
2. Edit products.json to add your own products
3. Replace "your-affiliate-id" with your actual Amazon affiliate ID
